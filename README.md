# Pablino
# Bitrix24 Knowledge Base Telegram Bot

Telegram-бот для работы с базой знаний Bitrix24 (поиск статей, обновление KB и т.д.).

## Описание

Бот позволяет пользователям быстро искать информацию в базе знаний Bitrix24 прямо из Telegram.
Основные функции:
- Поиск статей по ключевым словам
- Обновление базы знаний (через отдельный скрипт)
- ...
